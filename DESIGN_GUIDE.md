# Hungry Ninja — 网页设计风格指南

> 本指南基于餐厅实景图片提炼，用于网站开发时的视觉对齐。

---

## 1. 品牌定位

**风格关键词：** 日式居酒屋 × 街头活力 × 温馨随性  
**目标感受：** 进入网页就像走进一家热闹的日本小馆子 — 有烟火气、有趣味、不过分正式。

---

## 2. 配色方案

### 主色（Primary）
| 名称 | 色值 | 用途 |
|------|------|------|
| 灯笼红 | `#D42B2B` | 主要 CTA 按钮、高亮、Logo 配色 |
| 墨黑 | `#1A1A1A` | 导航栏背景、外观主色、深色区块 |
| 暖白 | `#FAF8F4` | 页面底色、卡片背景 |

### 辅色（Secondary）
| 名称 | 色值 | 用途 |
|------|------|------|
| 苔藓绿 | `#3D7A4F` | 标签、徽章、次要按钮 |
| 木质棕 | `#6B4226` | 分割线、装饰元素、暖色调块 |
| 藏青蓝 | `#1B2A4A` | 暖帘色调，深色次背景 |

### 装饰色（Accent）
| 名称 | 色值 | 用途 |
|------|------|------|
| 飞鱼籽橙 | `#F4854A` | 食物标签、价格高亮 |
| 浅粉樱花 | `#F5D0C8` | 柔和背景色块、女性向内容区 |
| 金色稻米 | `#D4A843` | 徽标描边、高级感装饰 |

### 使用原则
- 深色背景（墨黑）+ 灯笼红 CTA 是核心组合
- 避免使用过于冷色调的蓝色，整体偏暖
- 白色区域用暖白而不是纯白 `#FFFFFF`

## 3. 字体

### 推荐字体组合

```css
/* 标题：有力量感的衬线/无衬线混用 */
font-family: 'Noto Serif JP', 'Playfair Display', serif;

/* 正文：清晰可读的无衬线 */
font-family: 'Noto Sans JP', 'Inter', sans-serif;

/* 特效/标语：粗体黑体感 */
font-family: 'Black Han Sans', 'Anton', sans-serif;
```

### 字体等级

| 等级 | 场景 | 大小 | 字重 |
|------|------|------|------|
| Hero 标题 | 首屏大标语 | 64–80px | 900 (Black) |
| Section 标题 | 各区块标题 | 36–48px | 700 (Bold) |
| 卡片标题 | 菜品名称 | 20–24px | 600 (SemiBold) |
| 正文 | 描述文字 | 15–16px | 400 (Regular) |
| 标签/小字 | 价格、标签 | 12–13px | 500 (Medium) |

### 中日文处理
- 页面中出现日文汉字时，使用 `Noto Serif JP` 或 `Noto Sans JP` 保证渲染质量
- 品牌名 "Hungry Ninja" 全大写展示，字间距 `letter-spacing: 0.05em`

## 4. 图形语言 & 装饰元素

### 核心装饰图案（从实景提炼）

1. **青海波纹（Seigaiha）**  
   日式鱼鳞波浪纹，用于区块底部分割线、背景水印  
   → 颜色：暖白上用 `rgba(212,43,43,0.08)` 的淡红色

2. **红色灯笼**  
   作为图标、插图、或页面装饰元素出现  
   → 可用 SVG 灯笼图形在首屏或 About 区块点缀

3. **暖帘（Noren）分割**  
   区块之间的过渡可用类似暖帘的竖条纹 + 底部波浪收尾  
   → CSS 实现：`border-bottom: wavy` 或 SVG 路径

4. **樱花**  
   作为轻量点缀（花瓣散落效果），不过分使用  
   → 仅在特定季节活动或 Hero 区块使用

5. **传统地砖纹**  
   黑白几何花纹，可用于 Footer 背景或装饰性区域

### 图片风格
- 食物图片：**强光、高饱和、俯拍或 45° 角**，突出食材色彩
- 环境图片：**温暖色调、带灯笼/砖墙**，营造氛围
- 避免：冷白光、无人场景、过度修图的食物图

## 5. 组件设计规范

### 按钮

```
主按钮（CTA）：
- 背景：#D42B2B（灯笼红）
- 文字：#FAF8F4（暖白）
- 圆角：6px
- 内边距：12px 28px
- hover：背景加深至 #B02222，轻微上移 translateY(-2px)

次要按钮：
- 背景：transparent
- 边框：2px solid #1A1A1A
- 文字：#1A1A1A
- hover：背景 #1A1A1A，文字变暖白
```

### 菜品卡片
```
- 背景：#FAF8F4
- 圆角：12px
- 阴影：0 4px 20px rgba(0,0,0,0.08)
- 图片区域：占卡片高度 60%，object-fit: cover
- 标题：Noto Serif JP, 18px, Bold
- 价格：#D42B2B, 16px, SemiBold
- 标签（辣/推荐/新）：圆角 pill 形，对应颜色编码
```

### 导航栏
```
- 背景：#1A1A1A（墨黑）
- Logo：白色 + 灯笼红点缀
- 链接：暖白，hover 变灯笼红，带下划线动画
- 高度：64px
- 移动端：汉堡菜单，展开为全屏遮罩
```

### 标签 / Badge
```
推荐：#D42B2B 背景，白字
新品：#3D7A4F 背景，白字
辣：#F4854A 背景，白字
素食：#3D7A4F 描边，绿字
```

## 6. 布局原则

### 间距系统（8px 基准）
```
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 40px
2xl: 64px
3xl: 96px
```

### 页面结构
```
Hero（首屏）
  ↓ 青海波纹分割
特色菜/招牌 Section
  ↓ 暖帘式分割
菜单浏览（分类 Tab）
  ↓
关于我们（图文并排）
  ↓
Google 评价展示
  ↓
预订/地址 + Footer
```

### 响应式断点
```
移动端：< 768px（单列布局为主）
平板：768px – 1024px（2列）
桌面：> 1024px（3列菜品网格）
```

## 7. 动效原则

- **过渡时长：** 200–300ms，使用 `ease-out`
- **卡片 hover：** 轻微上移 `translateY(-4px)` + 阴影加深
- **图片：** 懒加载，淡入效果 `opacity: 0 → 1`
- **灯笼元素：** 可加轻微摇摆动画（`@keyframes sway`，±3°，4s infinite）
- **避免：** 过多闪烁、弹跳、或速度过快的动画

---

## 8. 语气 & 文案风格

- **标题：** 直接、有力、带点趣味（"新鲜现做，每一碗都是诚意"）
- **描述：** 简洁，突出食材，不堆砌形容词
- **中英混用：** 菜品名可保留英文原名 + 中文副标题
- **日语元素：** 在视觉上点缀少量日文汉字（不要全文日语），增加氛围感

---

## 9. 不要做的事（Red Lines）

- ❌ 不用纯白 `#FFFFFF` 背景，偏冷
- ❌ 不用蓝色或紫色系作主色
- ❌ 不用过于圆润可爱的字体（如圆体），不符合居酒屋气质
- ❌ 不用平铺无重点的菜单列表，要有视觉层次
- ❌ 不用低分辨率食物图，食物图是卖点核心
