## 1. 品牌概览

| 项目 | 内容 |
|------|------|
| 餐厅名 | Hungry Ninja（日本寿司馆） |
| 概念 | 现代工业风 × 日式居酒屋 × 潮玩忍者 |

## 2. 色彩系统

### 2.1 主色调

| 角色 | 色值 | HEX | 用法 |
|------|------|-----|------|
| 背景深色 | 炭黑 | `#1A1A1A` | 页面背景、Footer、深色区域 |
| 背景次深 | 深灰 | `#2A2A2A` | Card 背景、导航栏、Section 交替 |
| 品牌红 | 忍者红 | `#C0392B` | 按钮、强调色、重要标签、价格 |
| 红深色 | 深红 | `#8B1A1A` | 按钮 hover、标题装饰、分隔线 |
| 暖木色 | 米白 | `#F5F0E8` | 正文文字、浅色区域卡片背景 |
| 金色 | 芥末金 | `#D4A84B` | 星级评分、VIP 标识、特殊标记 |

### 2.2 辅助色

| 角色 | HEX | 用法 |
|------|-----|------|
| 文字主色（深色背景上） | `#F5F0E8` | 深色背景上的正文 |
| 文字次色（深色背景上） | `#B0A898` | 深色背景上的辅助文字 |
| 文字主色（浅色背景上） | `#2C2C2C` | 浅色背景上的正文 |
| 文字次色（浅色背景上） | `#6B6B6B` | 浅色背景上的辅助文字 |
| 边框/分割线 | `#3A3A3A` | 卡片边框、分割线 |
| 成功绿 | `#27AE60` | 营业中、已预约等状态 |
| 错误红 | `#E74C3C` | 错误提示、已满标识 |

### 2.3 渐变色

```
品牌渐变: linear-gradient(135deg, #C0392B, #8B1A1A)
金色渐变: linear-gradient(135deg, #D4A84B, #B8922E)
暗色渐变: linear-gradient(180deg, #1A1A1A, #2A2A2A)
```

## 3. 字体排版

### 3.1 字体选择

| 层级 | 字体 | 字重 | 回退 |
|------|------|------|------|
| 标题/Logo | `'Playfair Display'` | 700, 900 | serif |
| 英文正文 | `'Inter'` | 400, 500, 600, 700 | sans-serif |
| 日文/中文 | `'Noto Sans SC'` | 400, 500, 700 | sans-serif |

**字体加载方案**：使用 Google Fonts CDN 加载，fallback 到系统字体。

### 3.2 字号层级

```
移动端(Mobile)             桌面端(Desktop)
H1: 2rem (32px)            H1: 3rem (48px)
H2: 1.5rem (24px)          H2: 2.25rem (36px)
H3: 1.25rem (20px)         H3: 1.5rem (24px)
正文: 1rem (16px)          正文: 1.125rem (18px)
小字: 0.875rem (14px)      小字: 0.875rem (14px)
标签: 0.75rem (12px)       标签: 0.75rem (12px)
```

### 3.3 行高

- 标题: 1.2
- 正文: 1.6
- 导航链接: 1

## 4. 页面布局 & 组件

### 4.1 首页结构

```
┌─────────────────────────────────────────────┐
│  NAVBAR (sticky)                            │
│  Logo + 导航链接 + 在线预约按钮               │
├─────────────────────────────────────────────┤
│  HERO SECTION (全屏)                        │
│  背景图 + 餐厅名 + tagline + CTA 按钮        │
│  覆盖暗色半透明遮罩                           │
├─────────────────────────────────────────────┤
│  ABOUT SECTION                              │
│  餐厅简介 + 图片                             │
├─────────────────────────────────────────────┤
│  FEATURED DISHES                            │
│  3-4 张特色菜品卡片                          │
├─────────────────────────────────────────────┤
│  DAILY SPECIALS                             │
│  星期几→特价表格/卡片                         │
├─────────────────────────────────────────────┤
│  REVIEWS SECTION                            │
│  顾客评价轮播/网格                            │
├─────────────────────────────────────────────┤
│  LOCATION & HOURS                           │
│  地址 + 营业时间 + 地图嵌入                   │
├─────────────────────────────────────────────┤
│  FOOTER                                     │
│  版权 + 社交链接 + 导航                      │
└─────────────────────────────────────────────┘
```

### 4.2 导航栏 (Navbar)

- **背景**: `#1A1A1A`，半透明 (`rgba(26,26,26,0.95)`)
- **高度**: 移动端 60px，桌面端 70px
- **Logo**: 左侧，"HUNGRY NINJA" 用 Playfair Display 900，白色
- **链接**: 居中/右侧，Inter 500，hover 时品牌红下划线动画
- **CTA 按钮**: "在线预约" 按钮，品牌红背景，白色文字，圆角 8px
- **移动端**: hamburger 菜单展开，全屏遮罩 + 居中链接列表

### 4.3 Hero 区域

- **全屏高度**: 100vh，最少 600px
- **背景**: 餐厅外景/内景精选图，cover 填充
- **遮罩**: 线性渐变从底部向上 (`rgba(26,26,26,0.9)` → `rgba(26,26,26,0.4)`)
- **内容**: 居中
  - H1: "HUNGRY NINJA" / "日本寿司馆"
  - 副标题: "Authentic Japanese Sushi & More"
  - CTA: "浏览菜单" 按钮 + "在线预约" 按钮（两个按钮并列）
- **动画**: 内容从下往上 fade in（CSS animation）

### 4.4 卡片组件 (Dish Cards / Review Cards)

```
┌─────────────────┐
│  [菜品图片]       │  ← 16:9 裁切，hover 时轻微放大
├─────────────────┤
│  菜品名称         │  ← H3 白色
│  简短描述         │  ← 辅助文字色
│  ★★★★☆           │  ← 如有评价
│  $价格            │  ← 品牌红粗体
└─────────────────┘
```

- **背景**: `#2A2A2A`
- **圆角**: 12px
- **阴影**: `0 4px 20px rgba(0,0,0,0.3)`
- **间距**: 16px padding
- **hover**: transform translateY(-4px)，阴影加深
- **网格**: 移动端 1列，平板 2列，桌面 3-4列

### 4.5 每日特价 (Daily Specials)

表格形式或横向卡片轮播，显示星期一到星期日对应的特价菜品。

设计:
- 深色区域背景 (`#1A1A1A`)
- 每周七天用标签切换或全部显示
- 每个特价项：星期名（品牌红）+ 菜品名（白色）+ 原价（划线删除）+ 特价（金色）
- 响应式：移动端竖向排列，桌面端横向网格

### 4.6 顾客评价

- **风格**: 引用块风格，左侧引号装饰（品牌红）
- **星级**: 金色星星 ★
- **内容**: 评价文本（浅色） + 顾客名（辅助色）
- **布局**: 网格/轮播，2-3列

### 4.7 按钮样式

| 状态 | 主按钮 (Primary) | 次按钮 (Secondary) |
|------|-------------------|-------------------|
| 默认 | 品牌红背景 + 白色文字 | 透明 + 品牌红边框 + 品牌红文字 |
| Hover | 深红背景 | 品牌红背景 + 白色文字 |
| 圆角 | 8px | 8px |
| Padding | 12px 24px | 12px 24px |
| 字体 | Inter 600 | Inter 600 |

### 4.8 菜单页面

独立菜单页面 `/menu`：

- 左侧分类导航（固定/sticky）：Sushi Rolls、Sides、Sashimi、Desserts、Drinks
- 右侧内容区域
- 每个分类带标题分隔线
- 菜品项：名称（左）+ 价格（右），带简短描述
- 分类间有间距和标题装饰（红色分隔线 + 金色装饰点）
- 每日特价区域独立高亮

### 4.9 预约页面

独立预约页面 `/reserve`：

- 表单：日期选择器、时间选择器、人数、姓名、电话、邮箱、备注
- 深色卡片表单风格
- 提交按钮 → 弹出确认信息
- 表单验证：必填字段标记、格式校验
- 成功提交后显示感谢信息 + 预约编号

### 4.10 老板后台 (Admin)

独立路由 `/admin`：

- 简单登录页面（用户名 + 密码）
- 预约列表：表格展示所有预约记录
  - 字段：日期、时间、人数、姓名、电话、备注、状态（未处理/已联系/已完成）
  - 排序：默认按时间倒序
  - 操作：标记已处理、删除
- 响应式表格：移动端卡片布局

## 5. 响应式断点

```
Mobile:  < 768px    (1列布局)
Tablet:  768-1024px  (2列布局)
Desktop: > 1024px   (3-4列布局)
```

容器最大宽度: 1200px，居中 margin auto

---

## 6. 图像风格

- **菜品照片**: 暖色调、高饱和度、暗背景突出食物
- **餐厅环境**: 保留原有灯光氛围，不加过度滤镜
- **图标**: 使用 Font Awesome 或 Heroicons，outline 风格
- **Logo**: 使用忍者主题元素，可在页头/页脚重复出现
- **图标颜色**: 品牌红或白色（取决于背景）

---

## 7. 动画 & 过渡

| 元素 | 动画 |
|------|------|
| 页面载入 | 内容淡入 + 从下往上 (0.6s ease-out) |
| 卡片 hover | translateY(-4px) + 阴影加深 (0.3s) |
| 导航链接 hover | 下划线从中间展开 (0.2s) |
| 按钮 hover | 背景色过渡 (0.2s) |
| 滚动时 | 元素进入视口时淡入 (Intersection Observer) |
| 移动端菜单 | 从右滑入 (0.3s) |

## 8. 技术栈要求

| 技术 | 用途 |
|------|------|
| **前端框架** | React (Next.js 或 Vite) |
| **CSS** | Tailwind CSS |
| **数据库** | SQLite (通过 Prisma/Better-SQLite3) |
| **API** | Next.js API Routes 或 Express |
| **部署** | Replit 托管 |

### 数据库 Schema (预约系统)

```sql
-- bookings 表
id          INTEGER PRIMARY KEY AUTOINCREMENT
date        TEXT NOT NULL        -- '2026-06-15'
time        TEXT NOT NULL        -- '18:30'
guests      INTEGER NOT NULL
name        TEXT NOT NULL
phone       TEXT NOT NULL
email       TEXT
notes       TEXT
status      TEXT DEFAULT 'pending'  -- 'pending' | 'contacted' | 'completed'
created_at  DATETIME DEFAULT CURRENT_TIMESTAMP

-- admin 表
id          INTEGER PRIMARY KEY AUTOINCREMENT
username    TEXT UNIQUE NOT NULL
password    TEXT NOT NULL          -- bcrypt 加密
```

## 10. 设计参考

- **整体 vibe**: 暗色系日本餐厅官网（参考类似黑色+红色的寿司店网站）
- **字体搭配**: Playfair Display 做英文字标（优雅感），Inter 做正文（现代感）
- **忍者元素**: 可用 ninja 相关 icon 作为装饰点缀（如手里剑形状的分隔符）
- **氛围关键词**: 暗黑、温暖、潮流、正宗、日式
